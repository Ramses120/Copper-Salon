# Copper Salon

Beauty salon appointment web app with public site and admin dashboard.

## Features
- Public marketing site with services and portfolio
- Appointment booking flow
- Admin dashboard for services, staff, bookings, and promotions
- Supabase-backed data storage and auth

## Requirements
- Node.js 18+
- npm

## Setup
1. Install dependencies:
   
   npm install

2. Create your local env file:
   
   cp .env.local.example .env.local

3. Fill in the Supabase values in `.env.local`.

4. Run the dev server:
   
   npm run dev

## Deployment
- Configure the same environment variables in your hosting provider (Netlify/Vercel).
